## 乐玩APP

--------------------------------------------------------------------------------------------
本项目是在大三时所做的，和zx,wly等人一起开发，因为没有基础，所以一切都时从头做起，耗时六个多月
但是由于后期我们三个都要找工作，无暇顾及项目，因此这个项目就没有上线宣传
--------------------------------------------------------------------------------------------
项目服务端是运行在腾讯云服务器中的，采用php语言做业务逻辑处理并且与数据库进行交互
